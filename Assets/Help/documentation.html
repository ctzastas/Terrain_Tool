Documentation

 

Το συγκεκριμένο εργαλείο είναι για την δημιουργία προσαρμοσμένου Τerrain για την μηχανή παιχνιδιών Unity

Έκδοση: 2019.3.0f1

 

 

Περιεχόμενα

1. CREATE TERRAIN

2. RANDOM TERRAIN

3. MIDPOINT DISPLACEMENT

4. GENERATE & SAVE TEXTURE

5. LOAD TEXTURE

 

 

1. Create Terrain

Το πρώτο πράγμα που πρέπει να κάνουμε ανοίγοντας το πρότζεκτ είναι να πάμε στα Tools και μετά Custom Terrain

Θα μας εμφανιστεί ένα παράθυρο το οποίο θα έχει τις επιλογές που χρειαζόμαστε για να δημιουργήσουμε το Terrain

α. Προσθέτουμε τις επιθυμητές τιμές στους άξονες (x,y,z) και την ανάλυση που θέλουμε.

β. Προσθέτουμε ένα όνομα για το νέο Terrain.

γ. Πατάμε το κουμπί Create Terrain

Στη συνέχεια θα μας εμφανιστεί το Terrain στο Hierarchy το οποίο θα έχει αποθηκευτεί στον φάκελο Custom Terrain ο οποίος δημιουργήτε αυτόματα στα Assets του πρότζεκτ μας, όπως επίσης θα γίνει και αυτόματα η σύνδεση του Component Custom Terrain στο νέο Terrain.

Τέλος μέσα από το Component Custom Terrain μπορούμε να προσαρμόσουμε το Terrain μας με τις λειτουργίες που αναγράφονται παρακάτω.

 

 

2. Random Terrain Button

Το πρώτο κουμπί του εργαλείου μας δίνει την δυνατότητα να δημιουργήσουμε ένα τυχαίο Terrain με την αλλαγή των τιμών στους άξονες (x,y).

 

 

3. Mindpoint Displacement Button

Εδώ μπορούμε να προσαρμόσουμε ένα τυχαίο Terrain ρυθμίζοντας τα μεσαία σημεία κάθε άκρου στο μέσο όρο των δύο γωνιών που είναι μεταξύ τους, συν ή πλην ένα τυχαίο ποσό, αλλάζοντας πάλι τις τιμές στους άξονες (x,y) αλλά και ομαλοποιώντας τις κορυφές με την τιμή Roughness.

 

 

4. Generate & Save Button

Με αυτή την λειτουργία μπορούμε να δημιουργήσουμε μία εικόνα από το υπάρχον Terrain, που έχουμε φτιάξει με τα κουμπία Random Terrain ή Mindpoint Displacement που μας δέιχνει το ύψος του Terrain σε grayscale μορφή, πατώντας το κουμπί Generate Texture. Έπειτα μπορούμε να την αποθηκεύσουμε στον φάκελο Maps, ο οποίος δημιουργείτε αυτόματα με το πάτημα του κουμπιού Save Texture στα Assets του πρότζεκτ μας.

 

 

5. Load Button

Το τελευταίο κουμπί μας δίνει την δυνατότητα να φορτώσουμε τις εικόνες που έχουμε αποθηκεύση, με την λειτουργία Generate & Save και αν θέλουμε να αλλάξουμε και την κλίμακα που θα έχει. Πατώντας το Load Texture μας φορτώνει την εικόνα και την μετατρέπει σε Terrain
